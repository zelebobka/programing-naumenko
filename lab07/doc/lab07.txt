додаю #include <math.h> перед кодом.
2) Додаю у функцію for sqrt (n) для обчислення кореня n.
3) при компелювання clang -g main.c -o main.bin додаю в кінці -lm для правильної компіляції (clang -g main.c -0 main.bin -lm).
Код програми:
#include <math.h>
int main()
{
int n = 11;
int result;
for (int i = 2; i < sqrt(n); i++){
if ( n % i == 0)
result = 0;
break;
if ( n % i == 1)
result = 1;
}

return 0;
}
