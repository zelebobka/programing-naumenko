Основна частина:
1) створив змінну n-(number) над якою ми і проводитимемо наші операції.
2) створив змінну result.
3)з допомогою for задав функцію яка перевірятиме,
чи просте число.
4) через if виводжу результат.
5) якщо результат дорівнює нулю, то число натуральне не просте,
а якщо помилка означає, що число просте (помилка буде тільки при простому числі).
Код програми:
int main(){
int n = 7;
int result;
for (int i = 2; i < n; i++){
if ( n % i == 0)
result = 0;
break;
if ( n % i == 1)
result = 1;
}

return 0;
}
Результат:
$0 = 32767 - просте число
